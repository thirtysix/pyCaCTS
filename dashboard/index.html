pyCaCTS

Master transcription factors across the DepMap cell-line panel

A validated, fast Python reimplementation of CaCTS (Cancer Core Transcription factor Specificity), which scores how lineage-specifically each transcription factor is expressed. Here it is applied both to the DepMap/CCLE cell-line panel (lineage, disease, subtype, model type, and each cell line) and to TCGA tumors (tumor type, molecular subtype, sample type), so you can read out the candidate master regulators of any cancer group, and compare the tumor with its cell-line models. Scores match the original R to floating-point precision, computed thousands of times faster.

TF scores

Every transcription factor's CaCTS specificity score for a chosen group, sortable. The Data toggle switches between DepMap cell lines and TCGA tumors. Lower CaCTS score = more group-specific (rank 1 = most specific). Class (specific / non-specific) uses the top-5%-score ∩ top-5%-expression rule, and the two gates behind it are shown as their own columns. Each TF also carries its family, per-group CRISPR essentiality (Chronos; not at the single-cell-line level), cross-group breadth, and out-links. ⬇ TSV downloads the current sorted / filtered view.

Tumor vs model

For a cancer, its specific master TFs in the TCGA tumor beside those in the matched DepMap cell-line models. The shared set is the master-TF program the model preserves; tumor-only factors are lost in culture (often microenvironment / lineage TFs), model-only ones are gained. Specificity is scored independently in each panel (tumor across TCGA types, model across DepMap subtypes), so this compares which factors each calls a master regulator.

Within-cancer master TFs

Inside a single cancer, which master TFs distinguish its subgroups. CaCTS is re-scored with the reference set restricted to one cancer's own samples, across either its molecular subtypes (TCGA subtype working-group calls) or tumor vs adjacent-normal tissue. Each column is a subgroup, listing its most-specific expressed TFs ranked within that cancer. Because the whole cancer shares a lineage, the strict specific-MTF gate is sparse here, so each list is the top TFs by specificity among the well-expressed ones; S flags the few that still clear the strict rule. Breast is a clean read: LumA / LumB surface ESR1, FOXA1, GATA3.
